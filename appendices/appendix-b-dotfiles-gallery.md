## Annotated Dotfile Reading Guide
When studying a rice:
1. Start with the compositor config (hyprland.conf / sway config)
2. Find the bar config (shell.qml or waybar config)
3. Trace theme application (colors.json, stylix config, or manual CSS)
4. Note the startup sequence (exec-once / autostart)
5. Check the keybindings for workflow insights
